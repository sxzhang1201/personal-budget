# Workflows

Step-by-step procedures for day-to-day budget tracking.

| Workflow | When to use |
|----------|-------------|
| [Add expenses](add-expenses.md) | After each purchase or batch import |
| [Monthly summary](monthly-summary.md) | Quick category totals in the terminal |
| [Dashboard](dashboard.md) | Charts, trends, and richer metrics |

All workflows read from the same file: [`data/expenses.csv`](../where-data-lives.md#expenses-csv).

Budget month filtering is shared — see [Budget month rule](../rules/budget-month.md).
